# wanted_pre_onboarding

원티드 프리온보딩 백엔드 코스 지원 과제

## 기능 요구 사항

- 채용공고 읽기 / 등록 / 수정 / 삭제 API
  - 채용공고 리스트 API
  - 채용공고 디테일 API
    - 해당 회사가 올린 다른 채용공고 ID 리스트 포함
  - 채용공고 등록 API
    - 회사 ID, 포지션, 상세설명, 보상금, 사용기술 정보 포함
  - 채용공고 수정 API
    - 회사 ID 제외 전체 정보 수정 가능
  - 채용공고 삭제 API
    - DB에서 완전 삭제
- 키워드로 채용공고 검색 API
  - 텍스트 정보에 한해 검색 가능
- 채용공고 지원 API
  - 중복 지원 제한
